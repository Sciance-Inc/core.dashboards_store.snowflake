# Présentation

Le Store Snowflake est un projet commun. Il sert à partager du code *dbt* entre plusieurs centres de services scolaires.

Le projet vise les tableaux de bord de formation professionnelle. Il met en commun les transformations qui peuvent être partagées. Il laisse chaque organisation garder ses règles locales lorsque la réalité diffère.

## Pourquoi ce projet existe

Sans dépôt commun, chaque équipe réécrit les mêmes transformations. Les règles finissent par diverger. Les corrections sont difficiles à partager.

Avec le Store Snowflake, une correction utile à tous peut vivre dans le dépôt commun. Une règle propre à une organisation peut vivre dans le dépôt local.

## Résultat attendu

Le résultat attendu est un socle stable de tables pour les tableaux de bord de formation professionnelle.

Chaque équipe doit pouvoir lire le code, comprendre les règles et vérifier les résultats.

## Point de contrôle

Après cette page, le lecteur doit comprendre la différence entre le dépôt commun et le dépôt local.
