# Revue et collaboration

La revue sert à améliorer le changement avant intégration.

Elle protège les autres équipes.

## Ce que la revue vérifie

- exactitude métier;
- lisibilité;
- tests;
- contrat de données;
- impact sur les autres CSS;
- documentation.

## Comment répondre

Répondre aux commentaires avec des faits.

Mettre à jour le code ou expliquer pourquoi le changement proposé n'est pas retenu.

## Point de contrôle

Une discussion utile laisse une trace de la décision.
