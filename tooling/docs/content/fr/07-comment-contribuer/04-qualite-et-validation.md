# Qualité et validation

Un changement doit être validé avant fusion.

## Validation minimale

- formatage fait;
- exécution *dbt* ciblée;
- tests pertinents lancés;
- documentation mise à jour;
- aucun secret ajouté;
- impact décrit.

## Site de documentation

Le site doit être généré avant publication.

Les liens importants doivent être vérifiés.

## Point de contrôle

La *Pull Request* doit lister les commandes lancées.
