# Dashboards

Le dossier *dashboards* contient le code propre à un tableau de bord.

Il prépare les tables finales attendues par *Power BI*.

## Ce qui appartient au dossier dashboards

- tables finales;
- vues proches du modèle du rapport;
- indicateurs qui ne servent qu'à ce rapport;
- ajustements de présentation nécessaires au rapport.

## Point de contrôle

Si une table ne sert qu'à un tableau de bord précis, elle peut appartenir au dossier *dashboards*.
