# Checklists

## Installation

- *WSL2* fonctionne;
- *Python* est installé;
- *Poetry* fonctionne;
- les dépendances sont installées;
- `dbt debug` réussit.

## Nouvelle règle d'affaires

- la règle est écrite;
- le lieu du changement est choisi;
- les tests sont ajoutés;
- la documentation est mise à jour;
- la *Pull Request* explique l'impact.

## Production

- le bon *target* est utilisé;
- les tests passent;
- les droits sont vérifiés;
- le rapport pointe vers la bonne base;
- les règles sont validées.
