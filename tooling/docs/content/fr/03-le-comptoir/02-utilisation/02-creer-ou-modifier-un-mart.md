# Créer ou modifier un mart

Un *mart* sert lorsqu'une transformation est utile à plusieurs tableaux de bord.

## Quand créer un mart

Créer ou modifier un *mart* lorsque la règle est commune et réutilisable.

Exemples : population de référence, inscriptions, paiements, parcours.

## Étapes

1. Décrire la règle.
2. Identifier les sources.
3. Créer le modèle.
4. Ajouter les tests.
5. Documenter le grain et les colonnes.

## Point de contrôle

Si le modèle ne sert qu'à un seul rapport, il appartient peut-être au dossier *dashboards*.
