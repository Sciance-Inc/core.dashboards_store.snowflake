# Checklist avant production

Cette liste doit être relue avant une mise en production.

## Environnement

- le *target* est `prod`;
- la base cible est `store`;
- le rôle est le bon;
- l'entrepôt est contrôlé.

## Données

- les *seeds* sont chargés;
- les modèles ciblés ont été exécutés;
- les tests importants passent;
- le *stamper* indique une exécution récente.

## Revue

- la *Pull Request* est relue;
- les règles d'affaires sont documentées;
- la documentation est à jour.

## Point de contrôle

S'il manque un élément, corriger avant de publier.
