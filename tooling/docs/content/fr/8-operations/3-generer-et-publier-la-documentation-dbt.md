# Générer la documentation dbt

La documentation *dbt* montre le graphe technique du projet.

Elle complète ce site.

## Générer

```bash
dbt docs generate
```

Résultat attendu : *dbt* produit les fichiers de documentation.

## Consulter

```bash
dbt docs serve
```

Résultat attendu : la documentation technique est disponible en local.

## Point de contrôle

Avant une revue importante, vérifier que la documentation *dbt* reflète les derniers modèles.
