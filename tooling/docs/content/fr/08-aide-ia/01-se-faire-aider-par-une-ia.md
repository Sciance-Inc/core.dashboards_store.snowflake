# Se faire aider par une IA

Si vous êtes perdu, vous pouvez demander à une IA de lire la documentation avec vous.

L'IA peut expliquer les étapes. Elle ne remplace pas la revue, les règles TI ni la validation métier.

## Prompt à copier

```text
Lis la documentation à partir de https://docs.dashboards-store-snowflake.sciance.ca/llms.txt et aide-moi à configurer mon profil dbt Snowflake étape par étape. Pose-moi une question à la fois.
```

## Ne jamais partager

- mot de passe;
- clé privée;
- jeton;
- secret;
- données nominatives;
- extrait sensible de production.

## Point de contrôle

Si l'IA demande un secret, refuser et demander une autre méthode.
