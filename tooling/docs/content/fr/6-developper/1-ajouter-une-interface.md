# Ajouter une interface

Une *interface* doit rendre une source lisible par le Store.

Elle doit rester proche de la source.

## Étapes

1. Identifier la source.
2. Identifier les colonnes nécessaires.
3. Renommer les colonnes pour le Store.
4. Ajouter les tests de base.
5. Documenter le contrat de sortie.

## Point de contrôle

Une *interface* ne doit pas cacher une règle d'affaires importante.
