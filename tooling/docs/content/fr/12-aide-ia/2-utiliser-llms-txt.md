# Utiliser llms.txt

`/llms.txt` est un index de documentation pensé pour les assistants IA.

Il donne à l'IA le bon point de départ.

## Adresse officielle

```text
https://docs.dashboards-store-snowflake.sciance.ca/llms.txt
```

Si disponible, la version complète est :

```text
https://docs.dashboards-store-snowflake.sciance.ca/llms-full.txt
```

## Comment l'utiliser

Copier l'adresse dans l'IA.

Demander ensuite une aide pas à pas.

## Point de contrôle

L'IA doit citer les pages du site et rester dans le périmètre de la documentation.
