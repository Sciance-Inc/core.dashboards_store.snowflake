# Installer WSL2

*WSL2* permet d'utiliser Linux sur un poste Windows.

Les commandes du projet sont écrites pour un terminal Linux.

## Action

Installer *WSL2* selon la procédure de votre équipe TI.

Installer ensuite une distribution Ubuntu prise en charge par votre organisation.

## Vérification

Ouvrir un terminal Ubuntu et lancer :

```bash
uname -a
```

Résultat attendu : le terminal affiche une information Linux.

## Erreur fréquente

Si la commande n'est pas reconnue, vous n'êtes probablement pas dans le terminal Ubuntu.
