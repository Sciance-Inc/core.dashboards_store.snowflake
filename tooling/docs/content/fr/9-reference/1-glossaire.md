# Glossaire

## Core

Dépôt commun qui contient le code partagé.

## Dépôt local

Dépôt propre à un CSS. Il contient les adaptations locales.

## Interface

Couche qui stabilise les sources.

## Mart

Couche qui factorise le code utile à plusieurs tableaux de bord.

## Dashboard

Couche qui prépare les tables finales d'un rapport.

## Seed

Petit fichier `.csv` utilisé comme table de référence.

## Override

Remplacement local d'une ressource commune.

## Stamper

Mécanisme qui trace les tables produites par une exécution.
