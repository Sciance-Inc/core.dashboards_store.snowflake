# Conventions de nommage

Les noms doivent aider le lecteur à trouver le bon objet.

## Interfaces

Nommer une *interface* selon la source et l'objet lu.

## Marts

Nommer un *mart* selon le domaine commun qu'il prépare.

## Dashboards

Nommer un dossier *dashboard* selon le rapport qu'il alimente.

## Seeds

Nommer un *seed* selon la règle qu'il encode.

## Tags

Utiliser un *tag* pour regrouper les objets d'un même périmètre.

## Point de contrôle

Un nom doit être compréhensible sans ouvrir le fichier.
